# Wave Defense PvP

Ein Minecraft Fabric Mod für PvP-Training gegen KI-Bots.

## Features

### Arena Mode
- 7 verschiedene Kits: Mace, Sword, Axe, Bow, Crystal, UHC, Shield
- 3 Schwierigkeitsstufen: Easy, Medium, Hard
- Bots mit realistischem PvP-Verhalten (Strafing, W-Tapping, Jump Crits)
- Eigene PvP-Dimension

### Survival Arena
- Open-World Dimension mit Terrain
- Bots spawnen alle ~100 Blöcke
- Zufällige Kits für Bots und Spieler
- XP für besiegte Bots

### Lobby System
- Kit-Auswahl per Rechtsklick
- Schwierigkeitsauswahl per Shift+Klick
- Statistik-Tracking (Wins/Losses)

## Befehle

| Befehl | Beschreibung |
|--------|--------------|
| `/wd arena <kit> [difficulty]` | Starte Arena mit Kit |
| `/wd survival` | Starte Survival Arena |
| `/wd leave` | Verlasse Arena |
| `/wd exit` | Verlasse Survival Arena |
| `/wd lobby` | Teleport zur Lobby |
| `/wd kit <kit>` | Erhalte Kit-Items |
| `/wd stats` | Zeige Statistiken |

## Kits

- **Mace** - Wind Charge Combos, Smash-Angriffe
- **Sword** - W-Tap Combos, schnelle Angriffe
- **Axe** - Schild-Durchbruch, schwere Schläge
- **Bow** - Vorhersage-Pfeile, Kiting
- **Crystal** - Explosions-Schaden
- **UHC** - Angel-Combos
- **Shield** - Block + Counter

## Installation

1. Fabric Loader 0.15+ installieren
2. Fabric API installieren
3. Mod-JAR in den mods-Ordner legen

## Anforderungen

- Minecraft 1.21.x
- Fabric Loader 0.15+
- Fabric API
- Java 21+
